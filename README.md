# CodeBox.nvim

This is a simple neovim plugin that helps keeping blocks of code and reusing them as needed.

## Installation

Packer:
```
use('fabrizioperria/codebox.nvim',
    requires = {
            'plenary.nvim',
            'telescope'
        })
```

N.B.: Remember to call `require('codebox').setup({})`.
